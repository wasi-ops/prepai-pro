# PrepAI Pro

PrepAI Pro is an AI-powered resume analysis and interview preparation platform built using Python, OpenRouter LLM APIs, structured JSON outputs, validation pipelines, and interactive CLI workflows.

## Features

- Resume and job description analysis
- Job match score generation
- Matching and missing skills detection
- Personalized improvement suggestions
- Interview question generation
- Rubric-based answer evaluation
- Practice session history
- Score summary tracking
- JSON and text report generation

## Tech Stack

- Python
- OpenRouter API
- OpenAI Python SDK
- JSON
- Prompt Engineering
- VS Code
- Git and GitHub
